# Anthropic Claude Developer Credentials

## Purpose
Showcase of my completed Anthropic & Claude AI course certificates and technical credentials.

---

## Course Catalog & Purposes

### 🦾 Agentic AI & Developer Tools
1. **Claude Code 101**
   * *Course Purpose:* Learn how to use Claude Code effectively in your daily development workflow.
2. **Claude Code in Action**
   * *Course Purpose:* Apply Claude Code to practical, real-world development workflows and active repositories.
3. **Introduction to Claude Cowork**
   * *Course Purpose:* Learn to work alongside Claude on your real files and projects, covering the Cowork task loop, plugins, skills, research workflows, and responsible multi-step execution.
4. **Introduction to Agent Skills**
   * *Course Purpose:* Learn how to build, configure, and share Skills in Claude Code — reusable markdown instructions that Claude automatically applies to the right tasks at the right time.
5. **Introduction to Subagents**
   * *Course Purpose:* Learn how to use and create sub-agents in Claude Code to manage context, delegate tasks, and build specialized workflows that keep your main conversation clean and focused.

### 💻 Deep Tech & Integration Architecture
6. **Building with the Claude API**
   * *Course Purpose:* This comprehensive course covers the full spectrum of working with Anthropic models using the Claude API, from fundamental operations to advanced application architectural patterns.
7. **Introduction to Model Context Protocol**
   * *Course Purpose:* Learn to build Model Context Protocol (MCP) servers and clients from scratch using Python, mastering core primitives like tools, resources, and prompts to connect Claude with external services.
8. **Model Context Protocol: Advanced Topics**
   * *Course Purpose:* Discover advanced Model Context Protocol implementation patterns including sampling, notifications, file system access, and transport mechanisms for production MCP server development.

### 🧠 Core Foundations & Fluency
9. **Claude 101**
   * *Course Purpose:* Learn how to use Claude for everyday work tasks, understand core features, and explore resources for more advanced learning on other topics.
10. **AI Capabilities and Limitations**
    * *Course Purpose:* An introductory course detailing exactly how artificial intelligence works, what it can do, and what it cannot reliably do.
11. **AI Fluency: Framework & Foundations**
    * *Course Purpose:* Learn to collaborate with AI systems effectively, efficiently, ethically, and safely using core structural frameworks.
12. **Teaching AI Fluency**
    * *Course Purpose:* Empower academic faculty, instructional designers, and leaders to teach and assess AI Fluency in instructor-led settings.

### 🏢 Sector-Specific AI Fluency
13. **AI Fluency for Small Businesses**
    * *Course Purpose:* Empower small business owners and teams to leverage AI fluency to optimize localized operations, marketing loops, and business management.
14. **AI Fluency for Educators**
    * *Course Purpose:* Apply AI fluency concepts directly to teaching, learning, curriculum design, and modern educational assessment strategies.
15. **AI Fluency for Students**
    * *Course Purpose:* Empower students to develop AI Fluency skills that enhance learning, career planning, and academic success through responsible AI collaboration.
16. **AI Fluency for Nonprofits**
    * *Course Purpose:* Empower nonprofit professionals to develop AI fluency in order to increase organizational impact and efficiency while staying true to their mission and values.

### ☁️ Enterprise Cloud Architectures

    * *Course Purpose:* Master the deployment and integration of Claude models natively through AWS Bedrock cloud services, covering enterprise tool use, RAG, and secure scaling.
